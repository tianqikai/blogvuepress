# 7. 类加载与类加载器

##  7.1 一个类的生命周期（重点） 

类生命周期7个阶段 类从被加载到虚拟机内存中开始，到卸载出内存为止.  
它的整个生命周期包括：  
加载（Loading）、  
验证（Verification）、  
准备（Preparation）、  
解析（Resolution）、   
初始化（Initialization）、  
使用（Using）和  
卸载（Unloading）7 个阶段。  
<font color='red'><strong>其中验证、准备、解析 3 个部分统称为连接（Linking）</strong></font>
