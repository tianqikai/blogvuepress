# 1. 设置全局版本身份

```sh
git config --global user.name ‘tianqikai’
git config --global user.email ‘953811508@qq.com’
git config --global user.phone ‘18562328330’
```

这种方式在本地仓库中不设置本地用户时，默认使用的时全局身份
